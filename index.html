Sorting Algorithm Visualizer

Why this project?

To improve my data structure and algorithm knowledge. Was also a gateway into my next project so I can improve best move calculation speed.

Why tech stack?

After spending time with the Social Networking project, I was familar with react. I grew a liking to the framework and wanted to apply what I learned.

Challenges

Learning new algorithms, as well as timing the intervals with each interval. The biggest challenge was displaying the sorting in a timely matter. This allowed me to learn more about the inner workings of the algorithms.

What I would add

Colour any bar that the algorithm is looking at and colour the bars that are already sorted. As well as being able to sort other data sets than just integers.

Chess Game

Why this project?

At first, it was to test my vanilla javascript skills and make a game at the same time. Quickly into making the project, I thought about adding socket.io for players to play against eachother, as well as an AI the player can play against in either easy,medium, or hard mode. Found out very quickly I suck at chess.

Why tech stack?

To test my vanilla javascript knowledge since my previous two projects were written with React.

Challenges

Learning chess. The more I learned about chess the more complex the project became.

What I would add

Socket.io so two players can play against eachother. As well as a system that allows the user to gain points with each match they play, so they can earn enough points to purchase a chess set that they can play with. There are currently four chess sets for both black and white.
